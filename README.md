# Financial-Data-Playground

Este repositório atua como um laboratório prático para pesquisa e experimentação em modelagem quantitativa, análise de dados inter-regionais e engenharia financeira utilizando Python. 

A estrutura é segmentada em eixos temáticos com ênfase em dados do mercado financeiro e exploração extensiva usando Pandas, Numpy, entre outras bibliotecas fundamentais em Data Science e modelos Quants.

## 📂 Estrutura do Repositório

### 1. Finanças e Quant (`/Financas_e_Quant`)
Cadernos focados em backtesting, otimização e construção de modelos de risco em Quantitative Finance.

*   [`analise_volatilidade_series_temporais.ipynb`](./Financas_e_Quant/analise_volatilidade_series_temporais.ipynb): Exploração rigorosa da volatilidade, construção de premissas com séries temporais financeiras e identificação de retornos condicionados de mercado.
*   [`balanceamento_carteira.ipynb`](./Financas_e_Quant/balanceamento_carteira.ipynb): Casos de aplicação da Teoria de Portfólio, mapeando as alocações em fronteira eficiente para maximizar Sharp Ratios.
*   [`itau_quant_challenge.ipynb`](./Financas_e_Quant/itau_quant_challenge.ipynb): Aprofundamento metodológico quantitativo do desafio estratégico do Itaú, com construção de lógicas algorítmicas de precificação e análise de risco.
*   [`backtestpro_extensao.py`](./Financas_e_Quant/backtestpro_extensao.py): Algoritmo de automação desenvolvido para extensão/capacitação massiva de execuções de backtesting com lógicas financeiras sob-medida.

### 2. Tratamento de Dados e Séries (`/Tratamento_de_dados`)
Investigações estatísticas de bases estaduais, rotinas de saneamento de dados complexos (`data cleaning`) e exploração gráfica.

*   [`limpeza_dados_seguro_rural.ipynb`](./Tratamento_de_dados/limpeza_dados_seguro_rural.ipynb): Scripts de higienização de base massiva usando transformações complexas para extração, normalização de schema e limpeza de vazios estruturais.
*   [`analise_dados_estado.ipynb`](./Tratamento_de_dados/analise_dados_estado.ipynb) & [`analise_dados_goias.ipynb`](./Tratamento_de_dados/analise_dados_goias.ipynb): Data Science descritivo e preditivo sobre panoramas estaduais focados na extração de deficiências, distribuições sociodemográficas e padrões não lineares.
*   [`analise_exploratoria_graficos.ipynb`](./Tratamento_de_dados/analise_exploratoria_graficos.ipynb): Abordagem estritamente analítica com pacotes focais de Data Visualization para apresentação de sumários rápidos a stakeholders.
*   [`operacoes_groupby_pandas.ipynb`](./Tratamento_de_dados/operacoes_groupby_pandas.ipynb): Aplicações massivas da utilidade multidimensional `groupby`, pivots e funções agregadoras em DataFrames extensivos via Pandas.

## 🚀 Reprodutibilidade (Como Executar)

Inicie as ferramentas deste repositório em seu ambiente clonando-o diretamente e instalando suas dependências prévias contidas no arquivo base `requirements.txt`:

```bash
git clone https://github.com/NkNaite/Financial-Data-Playground.git
cd Financial-Data-Playground
pip install -r requirements.txt
```
